## API 接口字段映射

### 用户模块

#### GET /api/Account/profile
**返回字段**: UserInfo (见上文用户信息表)

#### GET /api/user/consignmentCoupons
**请求参数**:
- `page`: 页码 (默认1)
- `limit`: 每页数量 (默认50, 最大100)
- `status`: 状态过滤 (1=可用, 0=已使用)

**返回字段**:
- `list`: ConsignmentCouponItem[] (见寄售券信息表)
- `total`: 总记录数
- `available_count`: 可用数量

### 藏品模块

#### GET /api/CollectionItem/detail
**新增返回字段**:
- `supplier_name`: 供应商名称
- `tx_hash`: 存证指纹 (替换fingerprint)
- `asset_code`: 确权编号
- `type`: 详情类型 (market/my)

#### POST /api/collectionItem/bidBuy (盲盒预约)
**请求参数**:
- `session_id`: 场次ID (必填)
- `zone_id`: 价格分区ID (必填)
- `extra_hashrate`: 额外加注算力 (0-50)

**返回字段**: TradeReservation (见交易预约表)

### 寄售模块

#### POST /api/collectionItem/consign
**请求参数**:
- `user_collection_id`: 用户藏品记录ID
- `price`: 寄售价格

**返回字段**:
- `coupon_used`: 使用的寄售券数量
- `coupon_remaining`: 剩余寄售券数量
- `waive_type`: 免除类型
- `rollback_reason`: 回滚原因

## 数据格式规范

### 时间格式
- **时间戳**: Unix时间戳 (秒级)
- **日期字符串**: `YYYY-MM-DD HH:mm:ss`

### 金额格式
- **类型**: string 或 number
- **精度**: 保留2位小数
- **单位**: 元 (人民币)

### 手机号格式
- **规则**: 1[3-9]开头的11位数字
- **示例**: 13800138000

### 确权编号格式
- **格式**: `37-DATA-COLLECTION-00000001`
- **说明**: 前缀 + 序列号

### 存证指纹格式
- **类型**: MD5哈希值
- **长度**: 64位字符串
