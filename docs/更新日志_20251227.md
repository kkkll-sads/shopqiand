# 2025-12-27 更新日志

## ✨ 新增功能 (New Features)

### 1. 地址与支付表单增强
-   **省市区数据补齐**: 新增 `public/data/pca-code.json`，包含完整的中国省市区行政区划数据。
-   **动态地址选择**: `RegionPicker` 组件升级，支持异步加载本地 JSON 数据，减小首屏体积。
-   **银行选择增强**: `BankPicker` 组件新增 **搜索** 功能和 **手动输入** 模式，解决部分银行无法找到的问题。

### 2. API 深度集成与验证
-   **旧资产解锁**: 集成 `checkOldAssetsUnlockStatus` 和 `unlockOldAssets` 接口。
-   **重置支付密码**: 验证并确认 `resetPayPasswordBySms` 接口可用。
-   **寄售券**: 配置 `consignmentCoupons` 接口。

## 💄 UI/UX 优化 (Improvements)

### 1. 个人中心 (Profile)
-   **资产卡片**: 优化了资产卡片的排版，增加了内边距 (`px-8 py-6`)，减少了字段间的视觉拥挤感，使界面更加大气通透。

## 🐛 修复与维护 (Fixes & Maintenance)

-   **接口定义**: 全面更新 TypeScript 接口定义 (`services/collection.ts`, `services/user.ts`, `services/wallet.ts`) 以严格匹配后端 v2.5 API 文档。
-   **代码清理**: 移除了部分废弃的 API 调用参数（如 `bidBuy`的旧模式参数）。
-   **Bug 修复**: 修复了资产卡片中“总资产”说明弹窗无法获取公告的问题（添加了 `total_assets` 映射）。
-   **内容更新**: 更新了签到页面中“邀请好友奖励”的文案描述（改为“1.5 元 - 2.0 元 / 人”）。
