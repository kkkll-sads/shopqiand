# 流水日志表

前端资金流水相关接口与数据结构说明。

---

## 一、接口一览

| 接口 | 路径 | 说明 |
|------|------|------|
| 全部明细（列表） | `GET /Account/allLog` | 统一流水列表，支持类型、收支、时间、关键词筛选 |
| 合并流水子明细 | `GET /Account/allLogMergedItems` | 某条流水的合并子项（如多笔合并展示） |
| 资金明细详情 | `GET /Account/moneyLogDetail` | 单条流水详情（含 breakdown 等） |

---

## 二、全部明细列表 `allLog`

### 请求参数（GetAllLogParams）

| 参数 | 类型 | 必填 | 说明 |
|------|------|------|------|
| page | number | 否 | 页码 |
| limit | number | 否 | 每页条数 |
| type | string | 否 | 账户类型，见「账户类型 type」 |
| flow_direction | 'in' \| 'out' \| 'all' | 否 | 收支：收入/支出/全部 |
| start_time | string \| number | 否 | 开始时间 |
| end_time | string \| number | 否 | 结束时间 |
| biz_type | string | 否 | 业务类型，见「业务类型 biz_type」 |
| keyword | string | 否 | 关键词搜索 |
| token | string | 否 | 登录 token |

### 账户类型 type（与余额分类一致）

| value | 说明 |
|-------|------|
| all | 全部 |
| balance_available | 供应链专项金 |
| withdrawable_money | 可调度收益 |
| service_fee_balance | 确权金 |
| score | 消费金 |
| green_power | 绿色算力 |
| pending_activation_gold | 待激活确权金 |

### 响应（AllLogListData）

| 字段 | 类型 | 说明 |
|------|------|------|
| list | AllLogItem[] | 流水列表 |
| total | number | 总条数 |
| per_page | number | 每页条数 |
| current_page | number | 当前页 |

### 流水项 AllLogItem

| 字段 | 类型 | 说明 |
|------|------|------|
| id | number | 主键 |
| amount | number \| string | 变动金额（正负表示收支） |
| memo | string | 备注/说明 |
| createtime | number | 创建时间戳 |
| create_time | number | 创建时间（备用） |
| type | string | 账户类型（同 type 参数） |
| account_type | string | 账户类型（备用） |
| field_type | string | 字段类型（与 type 对应） |
| biz_type | string | 业务类型 |
| flow_no | string | 流水号 |
| before_value | number | 变动前余额 |
| after_value | number | 变动后余额 |
| remark | string | 备注（备用） |
| breakdown | object \| null | 子明细/拆分数据 |
| [key: string] | any | 其他扩展字段 |

---

## 三、资金明细详情 `moneyLogDetail`

### 请求参数

| 参数 | 类型 | 必填 | 说明 |
|------|------|------|------|
| id | number \| string | 与 flow_no 二选一 | 流水 ID |
| flow_no | string | 与 id 二选一 | 流水号 |
| token | string | 否 | 登录 token |

### 响应（MoneyLogDetailData）

| 字段 | 类型 | 说明 |
|------|------|------|
| id | number | 主键 |
| flow_no | string | 流水号 |
| batch_no | string | 批次号 |
| biz_type | string | 业务类型 |
| biz_id | number | 业务关联 ID |
| account_type | string | 账户类型 |
| amount | number | 变动金额 |
| before_value | number | 变动前余额 |
| after_value | number | 变动后余额 |
| memo | string | 备注 |
| create_time | number | 创建时间戳 |
| create_time_text | string | 创建时间文案 |
| title_snapshot | string | 标题快照（如商品名） |
| image_snapshot | string | 图片快照 |
| user_collection_id | number | 用户藏品 ID（业务关联） |
| item_id | number | 藏品 ID（业务关联） |
| breakdown | any | 子明细/拆分结构 |

---

## 四、合并流水子明细 `allLogMergedItems`

### 请求参数

| 参数 | 类型 | 必填 | 说明 |
|------|------|------|------|
| id | number \| string | 是 | 主流水 ID |
| flow_no | string | 否 | 流水号 |
| account_type | string | 否 | 账户类型 |
| token | string | 否 | 登录 token |

### 响应

| 字段 | 类型 | 说明 |
|------|------|------|
| list | AllLogItem[] | 合并后的子项列表 |
| total | number | 条数 |
| merge_scene | string | 合并场景 |
| merge_row_count | number | 合并行数 |

## 六、前端使用位置

| 页面/模块 | 说明 |
|-----------|------|
| AssetView（数字资产总权益） | 全部交易列表，调用 getAllLog，支持分类/收支/时间筛选 |
| AssetHistory | 历史流水页，getAllLog + 关键词 |
| MoneyLogDetail | 流水详情页，getMoneyLogDetail |
| OrderFundDetail | 订单资金详情，getAllLog 后前端过滤订单相关 |
| AssetHistoryLogCard | 合并流水子明细，getAllLogMergedItems |

接口与类型定义见：`src/services/wallet/logs.ts`、`src/constants/status-enums/finance.ts`、`src/constants/balanceTypes.ts`。
