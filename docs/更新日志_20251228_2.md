
## 2025-12-28 资产明细接口更新与绿色算力功能

### 新增功能
- **绿色算力 (Green Power)**: 
  - 新增 "绿色算力" 资产 Tab。
  - 支持查看绿色算力明细 (`type='green_power'`)。
  - 新增消费金兑换绿色算力功能，兑换比例 2:1。

### 优化与调整
- **资产明细接口 (AssetView)**:
  - 津贴/提现明细 (Tab 2) 调整为使用 `getAllLog` 接口，参数 `type='withdrawable_money'`，以支持查看所有可提现余额变动（含提现、收益等）。
  - 优化渲染逻辑，统一处理不同类型的资金流水格式 (`AllLogItem`)。

### API
- 新增 `/api/Account/exchangeScoreToGreenPower` (POST) 用于积分兑换算力。
